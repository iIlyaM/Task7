# Task7
18. Вводится массив целых чисел. Найти максимальную сумму подряд идущих элементов, образующих арифметическую прогрессию. Будем считать, что два любых элемента (или один для массива из одного элемента) всегда образуют такую прогрессию. Учесть, что прогрессия может быть как положительной, так и отрицательной. Также учесть, что элементы в массиве могут быть как положительные, так и отрицательные.
